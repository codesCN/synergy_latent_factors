# synergy_latent_factors

Code to compute synergies between latent factors.

- Case: resting-state fmri.
- Codes to compute Oinfo: https://github.com/danielemarinazzo/local_O_information_general
- Codes to create a violin plot: https://www.mathworks.com/matlabcentral/fileexchange/91790-al_goodplot-boxblot-violin-plot

# Reference: 
Ontivero-Ortega, M., Mijatovic, G., Faes, L., Rosas, F. E., Marinazzo, D., & Stramaglia, S. (2025). Localizing Synergies of Hidden Factors in Complex Systems: Resting Brain Networks and HeLa GeneExpression Profile as Case Studies. Entropy, 27(8), 820. https://www.mdpi.com/1099-4300/27/8/820
